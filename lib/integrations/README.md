# lib/integrations/

외부 API 어댑터.

예정 서브모듈:
- `google/` — Gmail, Google Calendar (OAuth, 최소 스코프).
- `search/` — 웹 검색 공급자.
- `weather/` — 날씨 공급자.
- `llm/` — LLM 공급자.
- `storage/` — Supabase Storage 래퍼 (필요 시).

규칙:
- 모든 외부 호출은 이 폴더를 경유한다.
- 토큰/시크릿은 환경 변수 또는 전용 저장소에서만 읽는다.
- 에러를 내부 도메인 에러 타입으로 매핑하여 상위 계층이 외부 API 세부를 몰라도 되게 한다.
- 레이트 리밋/재시도 정책을 어댑터 레벨에 둔다.

금지:
- 브라우저 자동화/로그인된 사이트 조작 어댑터.
- 메일 발송 어댑터 (정책 해제 전까지).
