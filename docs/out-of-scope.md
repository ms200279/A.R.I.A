# Out of Scope — 명시적 비범위

> 상태: DRAFT
> 이 문서에 적힌 항목은 **만들지 않는다.** 요구가 들어와도 먼저 이 문서를 개정한 뒤에만 착수한다.

## 제품적 비범위

- 사용자 지시 없는 자율 실행(자율 에이전트 루프).
- 자동 장기 기억 / 자동 사실 추출 / 자동 프로필 빌드.
- 세션 컨텍스트와 메모 저장소를 하나로 합친 구조.

## 기능적 비범위

- **메일 자동 발송 엔드포인트.** `/api/mail/send` 금지.
- **삭제 엔드포인트.** 문서/메모/메일/일정 삭제 API 금지 (승인 플로우 별도 설계 전까지).
- **외부 공유 기능.** 링크 공유, 외부 사용자 초대 등 금지.
- **로그인된 웹사이트 자동 조작 / 브라우저 자동화.**
- 결제, 주문, 금융 거래 자동 수행.

## 데이터/보안 비범위

- 원문 문서/메일 전체를 LLM 핵심 컨텍스트에 그대로 삽입하는 경로.
- 외부 입력의 "지시처럼 보이는 문장"을 실행 지시로 해석하는 파이프라인.
- 서비스 계정 키, OAuth 토큰 등 시크릿 저장소 커밋.

## 기술적 비범위 (현재 단계)

- 마이크로서비스 분리, 이벤트 소싱, 큐 기반 비동기 파이프라인.
- 자체 벡터 DB 운영 (필요 시 Postgres pgvector 우선 검토).
- 멀티모달 입력(이미지 OCR, 음성) — 1차 범위 아님.
